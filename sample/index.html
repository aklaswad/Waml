WAML - WebAudio Module Librarian

WAML is A Specification

propose to Web Audio Community that to make your code Reusable, Self-Documentable, and Sharable.

WAML is Very Simple Interface Spec

to make your code into WAML module. 
function ConstructorOfYourSynthesizer () { ... }

// then, just add this to the end of your *.js file (with some rules tho ;)
Waml.registerModule({
  name: "MySynthesizer",
  create: ConstructorOfYourSyntehsizer,
  isSynth: true
});

WAML Module behaves like native AudioNode

such like this. 
var wamlSynth = Waml.createModule('SYNTH_NAME');
var nativeDelay = ctx.createDelay();
wamlSynth.connect(nativeDelay);
nativeDelay.connect(ctx.destination);

var nativeOsc = ctx.createOsc();
var wamlEffect = Waml.createModule('EFFECT_NAME');
nativeOsc.connect(wamlEffect.inlet);
wamlEffect.connect(ctx.destination);

DEMOS

Load Module

You can easilly create module instance. 

var moduleInstance = Waml.createModule('Foobar');

Here, add selected module to the end of DSP chain.

Load Module from outer world on the fly

Also, you can load module from external javascript file on the fly. just push load button to load new module from outer world. ( or If you know any Waml-ized effector, put the URL to try it ;)

And then, you can see new module in the select menu above.
